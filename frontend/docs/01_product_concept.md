# 01. Product Concept 정의

## 1. 서비스 비전과 미션

### 비전 (Vision)
> "피부 건강을 몸무게처럼 매일 동일한 조건에서 측정하고, 데이터로 관리하는 세상"

체중계가 다이어트의 필수 도구이듯, 우리 서비스는 **피부 건강의 체중계** 역할을 한다. 매일 동일한 시간, 동일한 조건에서 피부 상태를 기록하고, 시간에 따른 변화를 데이터로 시각화하여 사용자가 자신의 피부를 과학적으로 이해하고 관리할 수 있게 한다.

### 미션 (Mission)
1. **측정의 일상화**: 매일 30초, 셀피 한 장으로 피부 상태를 기록
2. **효과의 객관화**: "이 화장품이 나한테 맞을까?"를 감이 아닌 데이터로 답변
3. **추천의 개인화**: 수만 명의 피부 데이터를 기반으로 "나와 비슷한 피부"에게 효과적이었던 제품을 추천

### 핵심 철학
- **No More Guessing**: 감에 의존하는 스킨케어에서 데이터 기반 스킨케어로
- **Your Skin, Your Lab**: 매일의 피부가 실험 데이터, 화장품이 실험 변수
- **Time Tells Truth**: 단기 사용감이 아닌, 4주·8주·12주 장기 데이터가 진짜 효과를 알려줌

## 2. 핵심 가치 제안 (Value Proposition)

### Primary Value: 화장품 효과의 데이터 기반 검증
| 기존 방식 | 우리 서비스 |
|-----------|------------|
| "이 크림 좋은 것 같아" (주관적 느낌) | 모공 크기 -12%, 피부결 점수 +8 (객관적 수치) |
| 리뷰 평점 4.5 (타인의 주관) | "나와 비슷한 피부 타입에서 효과 확인율 73%" |
| 1주 사용 후 판단 | 4주간 피부 지표 추이 그래프로 판단 |
| "피부과 가야 하나..." (불안감) | 피부 상태 악화 조기 감지 알림 |

### Secondary Value: 스킨케어 루틴 최적화
- **루틴 기록**: 현재 사용 중인 모든 제품과 사용 순서를 기록
- **변수 통제**: "이번 주부터 세럼을 바꿨더니" → 어떤 변수가 피부 변화에 영향을 줬는지 추적
- **A/B 테스트**: 화장품 A vs B를 2주씩 사용하며 데이터로 비교

### Tertiary Value: 피부 건강 인사이트
- 계절별 내 피부 패턴 파악
- 생활 습관(수면, 식단, 스트레스)과 피부 상태의 상관관계 발견
- 장기 피부 노화 추이 모니터링

## 4. 핵심 셀링 포인트 (Key Selling Points)

### 4.1 데이터 기반 화장품 효과 검증 ("Skincare Analytics")
**"4주 후, 이 세럼이 내 피부에 효과가 있었는지 숫자로 보여드립니다"**

- AI 피부 분석이 매일의 피부 상태를 **7가지 지표**(수분, 유분, 모공, 주름, 색소, 피부결, 트러블)로 수치화
- 화장품 사용 시작일을 기준으로 Before/After 비교 리포트 자동 생성
- 통계적 유의미성(Statistical Significance) 기반 "효과 있음/없음/판단 불가" 판정
- 사용 기간별(2주/4주/8주/12주) 효과 추이 시각화

### 4.2 개인 맞춤 피부 실험 ("Personal Skin Lab")
**"내 얼굴이 실험실, 내 루틴이 실험 프로토콜"**

- **실험 모드**: "이번 달에는 나이아신아마이드 세럼을 테스트합니다" → 기간 설정, 변수 통제 가이드 제공
- **대조 실험**: 좌/우 반얼굴 비교 테스트 기능 (Half-face Test)
- **루틴 변경 이벤트 자동 감지**: 새 제품 추가/제거 시점을 기준으로 피부 변화 분석
- **개인 성분 효과 프로필**: "나에게 효과적인 성분 TOP 5" 자동 도출

### 4.3 커뮤니티 데이터 기반 추천 ("Crowd-Verified Recommendations")
**"나와 피부가 비슷한 1,247명이 검증한 제품입니다"**

- 단순 리뷰가 아닌, **실제 피부 데이터 변화**를 기반으로 한 추천
- "나와 비슷한 피부 타입 + 비슷한 고민"을 가진 유저 그룹에서 효과가 검증된 제품 추천
- 제품별 **실제 효과 데이터 공개**: "이 제품을 4주 사용한 유저 중 68%에서 모공 축소 관찰"

### 5.2 시장 트렌드

1. **Skinimalism & "Skip-Care"**: 적은 수의 효과적인 제품만 사용하는 트렌드 → 제품 효과 검증 니즈 증가
2. **데이터 기반 의사결정**: MZ세대의 데이터 리터러시 향상, "데이터로 증명"에 대한 신뢰
3. **개인화(Personalization) 수요 폭발**: 획일적 추천이 아닌 "내 피부에 맞는" 솔루션 요구
4. **Skin Longevity**: "프리벤티브 에이징(Preventive Aging)" 트렌드, 20대부터 노화 예방에 투자
5. **뷰티테크 투자 확대**: 2023-2025년 뷰티테크 스타트업 투자 규모 연평균 24% 증가

### 5.4 핵심 기회 (Opportunity)

1. **Unmet Need**: "이 화장품이 나에게 효과가 있는가?"에 데이터로 답하는 서비스가 없음
2. **데이터 네트워크 효과**: 유저가 늘수록 추천 정확도 향상 → 자연 독점(Natural Monopoly) 가능성
3. **B2B 확장**: 축적된 피부 데이터를 화장품 브랜드에 인사이트로 제공 (Data-as-a-Service)
4. **플랫폼 확장**: 피부 → 두피 → 바디 → 전체 뷰티 건강 플랫폼으로 확장 가능
5. **커머스 연계**: 효과 검증된 제품의 직접 구매 연결 → 높은 구매 전환율 기대 (일반 뷰티 커머스 대비 3-5배)

## 6. 제품 원칙 (Product Principles)

1. **데이터가 답한다 (Data Speaks)**: 모든 추천과 판단은 데이터에 근거한다
2. **단순함이 지속을 만든다 (Simple is Sustainable)**: 매일 30초 이내로 완료되어야 한다
3. **신뢰는 투명성에서 온다 (Trust Through Transparency)**: AI 분석 과정과 근거를 투명하게 공개한다
4. **개인의 데이터는 개인의 것 (Your Data, Your Control)**: 데이터 활용에 대한 완전한 통제권을 사용자에게 부여한다
5. **장기적 가치를 추구한다 (Long-term Value)**: 즉각적 만족보다 4주, 12주, 1년의 가치를 설계한다
