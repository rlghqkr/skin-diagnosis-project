### 1.2 네비게이션 구조

```
┌─────────────────────────────────────────┐
│              Bottom Navigation           │
├────────┬────────┬────────┬──────┬──────┤
│  Home  │ Check  │Routine │Trend │ My   │
│  홈    │ 체크   │ 루틴   │트렌드│프로필│
│  🏠   │  📷   │  💧   │ 📊  │  👤 │
└────────┴────────┴────────┴──────┴──────┘

- 중앙 Check 탭: 가장 중요한 CTA, 강조 디자인 (원형, 브랜드 컬러)
- 활성 탭: 브랜드 컬러 + 채워진 아이콘
- 비활성 탭: 그레이 + 아웃라인 아이콘
```

## 2. 사용자 플로우 다이어그램

### 2.1 전체 사용자 여정 (User Journey)

```
[앱 진입]
    │
    ▼
[온보딩 - 최초 1회]
    │  피부 타입 설문 (건성/지성/복합성/민감성)
    │  알림 시간 설정 (아침/저녁 루틴 리마인더)
    │  첫 번째 피부 촬영 (기준 데이터)
    │
    ▼
[홈 화면] ◄──────────────────────────────────┐
    │                                         │
    ├──► [Daily Skin Check] ──► [분석 결과] ──┤
    │        │                                │
    │        └──► [이전 결과 비교]             │
    │                                         │
    ├──► [Skincare Routine] ──► [제품 기록] ──┤
    │        │                                │
    │        ├──► [제품 추가]                  │
    │        │      ├── 바코드 스캔            │
    │        │      ├── 텍스트 검색            │
    │        │      └── 수동 입력              │
    │        │                                │
    │        └──► [히스토리]                   │
    │                                         │
    ├──► [Skin Trend Dashboard] ──────────────┤
    │        ├── 종합 점수 그래프              │
    │        ├── 지표별 상세 뷰               │
    │        └── 제품 변경 마커               │
    │                                         │
    └──► [Profile / Settings] ────────────────┘
```

### 2.2 Daily Skin Check 상세 플로우

```
[Check 탭 진입]
    │
    ▼
[촬영 준비 안내]
    │  "동일한 조건에서 촬영해주세요"
    │  이전 촬영 조건 표시 (시간, 조명)
    │
    ▼
[카메라 활성화 + AR Guide 표시]
    │  얼굴 윤곽 오버레이
    │  거리 인디케이터
    │  조명 밝기 인디케이터
    │
    ├── 조건 미충족 → 실시간 가이드 피드백
    │
    ▼ (모든 조건 충족 시)
[자동 촬영 또는 수동 셔터]
    │
    ▼
[이미지 품질 확인]
    │
    ├── 품질 불량 → "다시 촬영하시겠습니까?"
    │
    ▼ (품질 통과)
[AI 분석 진행 (로딩)]
    │  프로그레스 바 + 분석 단계 표시
    │  "피부 수분 분석 중..."
    │  "모공 상태 확인 중..."
    │
    ▼
[분석 결과 화면]
    │  종합 피부 점수
    │  항목별 점수 (수분/탄력/모공/주름/색소)
    │  이모티콘 + 컬러 피드백
    │  이전 대비 변화
    │
    ├──► [상세 보기] → 각 항목 딥다이브
    ├──► [이전 비교] → 나란히 비교 뷰
    └──► [루틴 기록으로] → Routine Logging
```

### 2.3 Skincare Routine Logging 상세 플로우

```
[Routine 탭 진입]
    │
    ▼
[아침/저녁 토글]
    │
    ├── [아침 루틴]          [저녁 루틴]
    │      │                    │
    │      ▼                    ▼
    │  [등록된 제품 목록]   [등록된 제품 목록]
    │      │                    │
    │      ├── 체크박스 완료 표시
    │      ├── 순서 드래그 정렬
    │      └── 제품 탭 → 상세 정보
    │
    ▼
[+ 제품 추가]
    │
    ├── [바코드 스캔] → 카메라 → 제품 인식 → 확인/수정
    ├── [텍스트 검색] → 검색 결과 → 제품 선택
    └── [수동 입력]   → 브랜드/제품명/카테고리 입력
            │
            ▼
    [제품 카테고리 선택]
        토너(Toner) / 세럼(Serum) / 에센스(Essence)
        크림(Cream) / 선크림(Sunscreen) / 클렌저(Cleanser)
        마스크팩(Mask Pack) / 기타(Others)
            │
            ▼
    [루틴에 추가 완료]

[제품 교체 시]
    │
    ▼
[기존 제품 스와이프 → "교체" 버튼]
    │
    ▼
[새 제품 선택] → [교체 사유 선택 (선택사항)]
    │  "다 써서" / "효과 없어서" / "피부 트러블" / "기타"
    │
    ▼
[히스토리에 자동 기록]
    "2026-03-07: 제품A → 제품B 교체"
```

### 6.2 마이크로 인터랙션 (Micro-interactions)

```
1. 촬영 완료 시
   ┌──────────┐
   │  ✓ 촬영   │ → 0.3s pulse animation
   │  완료!    │ → 햅틱 피드백 (짧은 진동)
   └──────────┘

2. 루틴 체크 완료 시
   ☐ → ☑ (0.2s scale bounce)
   완료율 프로그레스 바 증가 애니메이션

3. 점수 변화 표시
   숫자 카운트업 애니메이션 (0 → 78, 1s duration)
   변화량 (+3) 슬라이드인 애니메이션

4. 제품 교체 시
   기존 제품 카드: 좌측 슬라이드 아웃
   새 제품 카드: 우측에서 슬라이드 인
   히스토리에 타임라인 노드 추가 애니메이션

5. 그래프 데이터 포인트 탭
   해당 포인트 확대 + 툴팁 표시
   다른 포인트는 흐리게 처리

6. 기간 필터 전환
   그래프 모핑 애니메이션 (기존 → 새 데이터)
   X축 라벨 페이드 전환

7. AR 가이드 피드백
   조건 충족: 초록 → 살짝 glow effect
   조건 미충족: 빨강 → 가이드 방향 애니메이션 표시

   스캔 프레임: 초록색 전환 + 체크 아이콘
   제품 정보: 하단에서 슬라이드업

```

## 8. 사용자 시나리오

### 시나리오 1: 첫 사용자 온보딩 + 첫 체크

```
[배경]
이름: 김지현 (28세, 여성)
피부 타입: 복합성, 수분 부족 고민

[플로우]

1. 앱 다운로드 후 첫 실행
   → 온보딩 스플래시 (3페이지 스와이프)
   → "피부 타입 간단 설문" (4문항)
   → 결과: "복합성 피부, 수분 관리 집중 추천"

2. 알림 설정
   → "아침 루틴 리마인더: 07:00"
   → "저녁 루틴 리마인더: 21:00"
   → "피부 체크 리마인더: 08:00"

3. 첫 번째 피부 촬영
   → Check 탭 진입
   → 촬영 가이드 안내 확인
   → AR 가이드 활성화
   → 거리/조명/각도 조건 맞춤
   → 자동 촬영 (3초 카운트다운)
   → AI 분석 진행 (약 10초)
   → 결과: 종합 68점
     - 수분: 55 (😐 관리 필요)
     - 탄력: 72 (🙂 보통)
     - 모공: 70 (🙂 보통)
     - 주름: 85 (😊 좋음)
     - 색소: 65 (🙂 보통)
   → "기준 데이터가 기록되었습니다!"

4. 현재 루틴 등록
   → Routine 탭 진입
   → 아침 루틴 제품 5개 등록 (검색으로)
   → 저녁 루틴 제품 6개 등록 (바코드 스캔)
```

### 시나리오 3: 제품 교체 후 트렌드 확인

```
[배경]
사용 60일차, 토너를 교체한 지 2주

[플로우]

1. Trend 탭 진입
   → 1M (1개월) 필터 선택
   → 종합 점수 그래프 확인

2. 그래프에서 제품 변경 마커 (🔺) 발견
   → 마커 탭
   → "2/10 토너 교체: 아이소이 → 라운드랩"
   → "교체 후 변화: 수분 +12점, 색소 +4점"

3. 수분 상세 뷰 진입
   → 💧 수분 카드 탭
   → Hydration Score Timeline 확인
   → 토너 교체 시점 이후 뚜렷한 상승 확인
   → AI 인사이트: "라운드랩 독도 토너가 수분 유지에
     효과적입니다. 현재 루틴을 유지하세요."

4. 피부 개선 신호 확인
   → ✅ "수분 점수 7일 연속 상승 중!"
   → ✅ "토너 교체 후 14일, 수분 +12점 개선"
   → 사용자 만족 → 현재 루틴 유지 결정

5. 비교 뷰에서 Before/After 확인
   → 2주 전 사진과 오늘 사진 나란히 비교
   → 슬라이더로 오버레이 확인
   → 변화를 시각적으로 확인
```

### 시나리오 4: 피부 트러블 발생 시

```
[배경]
새 세럼 추가 후 3일, 모공 점수 하락

[플로우]

1. Daily Check 결과 확인
   → 모공 점수: 65 (이전 72, ▼-7)
   → 😐 "관리가 필요해요"

2. Trend 탭에서 확인
   → 모공 상세 뷰 진입
   → 세럼 추가 시점 이후 하락 확인
   → ⚠ "세럼 추가 후 모공 점수 하락"

3. 대응 결정
   → Routine 탭에서 해당 세럼 스와이프
   → "삭제" 또는 "교체" 선택
   → 교체 사유: "피부 트러블"
   → 히스토리에 기록

4. 이후 트렌드 모니터링
   → 세럼 제거 후 모공 점수 회복 추이 확인
   → 제품 변경 마커에 기록 유지
```
